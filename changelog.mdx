<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## Upgrade Instructions

Follow these steps to upgrade Wildlife Studios seamlessly:

<Steps>
  <Step title="Backup Data" icon="database">
    Export your documents and settings.

````bash
curl -X POST https://api.example.com/v1/export \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{"format": "zip"}'
````
  </Step>

  <Step title="Update SDK" icon="package">
    Install the latest client library.

    <CodeGroup tabs="JavaScript,Python">
````javascript
npm install wildlife-studios@1.2.0
````

````python
pip install wildlife-studios==1.2.0
````
    </CodeGroup>
  </Step>

  <Step title="Restart Services" icon="refresh-cw">
    Redeploy your application to apply changes.
  </Step>
</Steps>
