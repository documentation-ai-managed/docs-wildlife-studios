## Prerequisites

Before you begin, ensure you have:

- A valid email address for account verification
- Access to a modern web browser like Chrome or Firefox
- Optional: An API key from your Wildlife Studios account for programmatic searches

<Callout kind="info">

Wildlife Studios requires email verification to ensure secure access to legal resources. Complete this step immediately after signup.

</Callout>

## Create Your Account

Follow these steps to set up your Wildlife Studios account.

<Steps>
  <Step title="Visit the Signup Page" icon="globe">
    Navigate to `https://dashboard.example.com/signup` and enter your email address and a strong password.
  </Step>
  <Step title="Verify Your Email" icon="mail">
    Check your inbox for a verification email from Wildlife Studios. Click the link to activate your account.
  </Step>
  <Step title="Complete Profile Setup" icon="user">
    Return to the dashboard and add your name and organization details to unlock full knowledge base access.
  </Step>
</Steps>

## Perform Your First Legal Search

Start with a basic keyword search for "GDPR compliance requirements".

### UI Search

Use the dashboard search bar directly.

### API Search

For programmatic access, use the search endpoint.

<CodeGroup tabs="JavaScript,Python,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/search', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      query: 'GDPR compliance requirements',
      limit: 10
    })
  });
  const results = await response.json();
  console.log(results);
  ```
  ```python
  import requests

  response = requests.post(
      'https://api.example.com/v1/search',
      headers={
          'Authorization': 'Bearer YOUR_API_KEY',
          'Content-Type': 'application/json'
      },
      json={
          'query': 'GDPR compliance requirements',
          'limit': 10
      }
  )
  results = response.json()
  print(results)
  ```
  ```bash
  curl -X POST https://api.example.com/v1/search \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "query": "GDPR compliance requirements",
      "limit": 10
    }'
  ```
</CodeGroup>
